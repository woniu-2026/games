# 🐍 贪吃蛇 (Snake Game)

一款基于 Canvas 的经典贪吃蛇网页小游戏，支持关卡升级、障碍物、金色食物和粒子特效。

## 快速开始

直接用浏览器打开 `snake.html` 即可游玩，无需安装任何依赖。

## 操作方式

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` / `W A S D` | 控制蛇的移动方向 |
| `Space` | 暂停 / 继续 |

## 游戏特性

- **经典玩法** — 控制蛇吃掉红色食物，每吃一个 +10 分
- **金色食物** ★ — 有概率随机出现，限时 7 秒，吃掉 +50 分
- **关卡升级** — 每吃 `6 × 当前等级` 个食物升一级，速度加快，并新增障碍物
- **障碍物** — 每升一级增加 2 个，撞上即游戏结束
- **穿墙机制** — 蛇可以从边界穿到对侧
- **粒子特效** — 吃到食物和金色食物时有彩色粒子爆炸效果
- **最高分存档** — 最高分会保存在浏览器的 `localStorage` 中

## 技术细节

- 纯 HTML + CSS + JavaScript，单文件无依赖
- 20×20 网格，Canvas 渲染
- 使用 `requestAnimationFrame` 做渲染循环，`setInterval` 控制游戏逻辑帧率
- 蛇身渐变色 + 圆角矩形绘制，蛇头带眼睛
- 食物带有呼吸脉冲动画
